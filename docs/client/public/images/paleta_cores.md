# Paleta de Cores - Site André Duarte

## Cores Identificadas na Marca

### Cores Principais
1. **Bege/Nude Claro** (#E8D5C4 aproximadamente)
   - Tom suave, acolhedor
   - Transmite calma e neutralidade
   - Uso: Backgrounds, seções alternadas

2. **Azul Escuro/Navy** (#4A5168 aproximadamente)
   - Tom profissional e confiável
   - Transmite seriedade e estabilidade
   - Uso: Títulos, botões, elementos de destaque

3. **Roxo/Lilás Médio** (#8B7E9B aproximadamente)
   - Tom intermediário, equilibrado
   - Transmite criatividade e empatia
   - Uso: Elementos secundários, ícones, destaques

### Cores Complementares (do site atual)
4. **Branco** (#FFFFFF)
   - Limpeza visual
   - Espaçamento e respiro

5. **Cinza Claro** (#F5F5F5)
   - Backgrounds sutis
   - Separação de seções

6. **Verde Suave** (das plantas/elementos naturais)
   - Toque de vida e crescimento
   - Uso pontual em detalhes

## Aplicação no Design

### Hierarquia Visual
- **Primária**: Azul Navy (CTAs, títulos principais)
- **Secundária**: Roxo/Lilás (subtítulos, ícones)
- **Background**: Bege claro, branco, cinza claro
- **Acentos**: Verde suave (opcional)

### Psicologia das Cores para Psicologia
- Bege: Acolhimento, segurança, neutralidade
- Azul: Confiança, profissionalismo, calma
- Roxo: Empatia, criatividade, transformação
- Verde: Crescimento, renovação, esperança

## Contraste e Acessibilidade
- Texto escuro (navy) sobre fundos claros (bege/branco)
- Garantir contraste mínimo WCAG AA (4.5:1)
- Botões com cores sólidas e legíveis
